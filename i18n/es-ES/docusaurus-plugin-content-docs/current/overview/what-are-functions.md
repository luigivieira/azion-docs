# Qué son las Functions

Las Azion Edge Functions son funciones serverless que se ejecutan en la red de borde de Azion, cerca de tus usuarios. Permiten ejecutar lógica personalizada en el borde — sin gestionar servidores, con latencia casi nula y sin cold starts.

Esta página es un marcador de posición y se ampliará con documentación detallada.
