# Qué es una Function Instance

Una **Function Instance** es una referencia configurada a una Edge Function dentro de una Edge Application. Vincula el código de una función específica a una aplicación, permitiendo que el Rules Engine la invoque bajo condiciones definidas.

Esta página es un marcador de posición y se ampliará con documentación detallada.
