# Flujo de Procesamiento de Solicitudes

El flujo de procesamiento de solicitudes en la plataforma Azion sigue esta secuencia:

    Solicitud del Cliente
        ↓
    Edge Node
        ↓
    Aplicación
        ↓
    Rules Engine
        ↓
    Function Instance
        ↓
    Ejecución de la Function
        ↓
    Origen / Storage / Respuesta

Esta página es un marcador de posición y se ampliará con documentación detallada.
