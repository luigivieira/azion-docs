# Requisitos Previos

Antes de comenzar, asegúrese de tener lo siguiente:

- Una **cuenta de Azion**. Si no tiene una, [cree su cuenta](https://console.azion.com/signup) — es gratis para comenzar.
- Acceso a la **Consola de Azion**, la interfaz basada en web donde creará y gestionará sus funciones.

:::info No se requiere configuración local
No necesita instalar nada en su máquina para esta guía. Todo se ejecuta directamente en el navegador a través de la Consola de Azion.
:::

## Qué construirá

En esta guía de Inicio Rápido, desplegará una pequeña Edge Function que responde a las solicitudes HTTP con una tarjeta de Pokémon aleatoria — obteniendo datos en vivo de [PokéAPI](https://pokeapi.co/) y devolviendo una respuesta HTML, todo en el borde.

Al final, habrá pasado por el flujo completo de despliegue:

1. Escribir y guardar una función
2. Crear una Edge Application
3. Exponerla a través de un Workload
4. Acceder a ella a través de un dominio público
5. Observar los logs de ejecución
