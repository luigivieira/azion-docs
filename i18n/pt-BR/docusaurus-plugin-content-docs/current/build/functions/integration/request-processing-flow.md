# Fluxo de Processamento de Requisições

O fluxo de processamento de requisições na plataforma Azion segue esta sequência:

    Requisição do Cliente
        ↓
    Edge Node
        ↓
    Aplicação
        ↓
    Rules Engine
        ↓
    Function Instance
        ↓
    Execução da Function
        ↓
    Origem / Storage / Resposta
