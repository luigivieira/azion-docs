# Pré-requisitos

Antes de começar, certifique-se de ter o seguinte:

- Uma **conta na Azion**. Se ainda não tiver uma, [crie sua conta](https://console.azion.com/signup) — o início é gratuito.
- Acesso ao **Azion Console**, a interface web onde você criará e gerenciará suas functions.

:::info Nenhuma configuração local necessária
Você não precisa instalar nada em sua máquina para este guia. Tudo é executado diretamente no navegador pelo Azion Console.
:::

## O que você vai construir

Neste guia de Getting Started, você vai fazer o deploy de uma pequena Edge Function que responde a requisições HTTP com um card aleatório de Pokémon — buscando dados em tempo real da [PokéAPI](https://pokeapi.co/) e retornando uma resposta HTML, tudo no edge.

Ao final, você terá passado por todo o fluxo de deploy:

1. Escrevendo e salvando uma function
2. Criando uma Edge Application
3. Expondo-a por meio de um Workload
4. Acessando-a por um domínio público
5. Observando os logs de execução
