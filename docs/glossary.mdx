  <h3 id="cold-start">Cold Start</h3>
  <p>The initialization latency that occurs when a function is invoked for the first time on an edge node, or after a period of dormancy. The runtime must spin up an isolate before handling the request.</p>

<div className="glossary-term" data-term="edge application">
  <h3 id="edge-application">Edge Application</h3>
  <p>An application built, delivered, and secured at the edge of the network using Azion's platform components.</p>
  <p>
    <small>
      <Link to="/platform-integration/functions-in-platform-architecture">First mentioned in: Functions in Platform Architecture</Link>

<div className="glossary-term" data-term="edge firewall">
  <h3 id="edge-firewall">Edge Firewall</h3>
  <p>A programmable and extensible security layer protecting applications directly at the edge nodes.</p>
  <p>

      <Link to="/platform-integration/functions-in-platform-architecture">First mentioned in: Functions in Platform Architecture</Link>

</div>

<div className="glossary-term" data-term="edge function">
  <h3 id="edge-function">Edge Function</h3>
  <p>A piece of programmable logic running on Azion Edge Nodes with very low latency, serving requests intimately close to end-users.</p>
  <p>
    <small>
      <Link to="/platform-integration/application-integration">First mentioned in: Application Integration</Link>
    </small>
  </p>
  <hr />
</div>

<div className="glossary-term" data-term="edge node">
  <h3 id="edge-node">Edge Node</h3>
  <p>A globally distributed server location within the Azion network that processes requests directly at the edge, rather than routing to a central origin server.</p>
  <p>
    <small>
      <Link to="/advanced/performance-optimization">First mentioned in: Performance Optimization</Link>

  <p>The processing stage in an Edge Application during which an incoming client request is handled, usually manipulated before reaching the origin.</p>
  <p>

  <h3 id="response-phase">Response Phase</h3>

  <p>
    <small>
      <Link to="/platform-integration/application-integration">First mentioned in: Application Integration</Link>
